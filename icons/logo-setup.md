# Extension Logo Setup

This folder contains the logos for the AI Site Collector Extension.

## Logo Files:
- `icon-16.png` - 16x16 pixels (browser toolbar)
- `icon-48.png` - 48x48 pixels (extension management page)
- `icon-128.png` - 128x128 pixels (Chrome Web Store)

## Logo Design:
The logo features:
- Modern AI theme with neon glow effects
- Purple to blue gradient background
- Central "AI" text in white
- Circuit board connections around the center
- Folder with documents in the background
- Perfect for AI website collection concept

## Usage:
These logos are automatically used in:
1. Browser toolbar (16px)
2. Extension popup (various sizes)
3. Extension management page (48px)
4. Chrome Web Store listing (128px)
5. Options page branding

All sizes have been generated maintaining the same design integrity across different resolutions.
